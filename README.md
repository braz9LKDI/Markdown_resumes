# Markdown Resumes

This repository contains my personal resume source files, built with [`markdown-resume`](https://github.com/junian/markdown-resume). If you want to see how they look, you can use the [online editor](https://www.junian.dev/markdown-resume) to preview them, just make sure to copy and replace the styles (with the ones in `style.css`).

If you are interested in my previous polished LaTeX template, you can find it in this [repository](https://github.com/braz9LKDI/LaTex_resume).

I chose this tool because I plan to maintain different resumes for different roles, and I want the workflow to stay as simple as possible. LaTeX is powerful, but its compilation artifacts can get noisy quickly, especially when managing multiple resume versions.
